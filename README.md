<h2>Controls</h2>

<h3>Movement</h3>

<ul>
  <li>A - Left</li>
  <li>D - Right</li>
  <li>Spacebar - jump</li>
  <li>Left shift - dash</li>
  <li>C - Attack</li>
</ul> 

<h3>Debbuging tools</h3>

<ul>
  <li>F1 - Restart Game</li>
  <li>F2 - Restart level</li>
  <li>F5 - Quicksave</li>
  <li>F6 - Load</li>
  <li>F9 - Show Colliders</li>
  <li>F10 - Godmode</li>
   <li>F11 - Cap Fps</li>
</ul>

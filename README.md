<h1>Bobobo's bizarre adventure</h1>

<h2>Controls</h2>

<h3>Movement</h3>

<ul>
  <li>A - Left</li>
  <li>D - Right</li>
  <li>Spacebar - jump</li>
  <li>Left shift - dash</li>
</ul> 

<h3>Debbuging tools</h3>

<ul>
  <li>F1 - Restart Game</li>
  <li>F2 - Restart level</li>
  <li>F3 - Change level </li>
  <li>F5 - Quicksave</li>
  <li>F5 - Load</li>
  <li>F10 - Godmode</li>
</ul>
